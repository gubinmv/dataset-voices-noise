База данных голосов и шумов

Из свободно распространяемого интернет ресурса http://audiobooks.pw были выбраны аудиокниги общей длительностью 266 часов, 16 мужских и 16 женских голосов. Голоса нужно было сравнить между собой, чтобы они оказались легко различимы между собой, т.е. не было двух похожих друг на друга голосов. Выбранные аудиокниги содержат только голоса чтецов и не содержат никах других звуков. Все фрагменты конвертированы в формат WAV с частотой дискретизации 8 кГц и глубиной звука 16 бит.

Треки шумов отобраны из датасета FreeSound и разбиты на фрагменты длительностью 5 секунд. Для описания фрагментов в базе данных используется описание каждого фрагмента, которое хранится в файле формата CSV. Формат описания фрагментов следующий: номер описания, имя описываемого файла описываемого фрагмента, вид (класс) шума, длительность в секундах, имя файла источника. Все фрагменты конвертированы в формат WAV с частотой дискретизации 8 кГц и глубиной звука 16 бит.

Формат описания фрагментов моей базы данных шумов:
•	номер описания,
•	имя описываемого файла описываемого фрагмента,
•	вид (класс) шума,
•	длительность в секундах,
•	имя файла источника.

Формат описания фрагментов базы данных FreeSound:
•	номер описания,
•	имя файла источника,
•	длительность в секундах,
•	описание файла из датасета FreeSound.

Описание структуры каталога базы данных шумов:
•	readme.txt – описание базы данных
•	dataset.csv – описание и расположение звуковых фрагментов 
•	source.csv – описание фрагментов источников с датасета FreeSound
•	dataset – каталог звуковых фрагментов
•	source – каталог файлов источников
•	create_dataset.py – python file, создание датасета из источников
•	print_dataset.py – python file, описание параметров датасета
